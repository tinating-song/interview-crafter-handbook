## 第三步：获取购买链接

创建完成后，面包多会提供一个购买链接，格式类似：
```
https://www.mianbaoduo.com/o/buy/xxxxx
```

## 第四步：更新README

将README.md中的购买链接替换为实际链接：

```bash
# 在项目中查找并替换
cd ~/interview-crafter-handbook
sed -i '' 's|https://example.com/buy|你的实际购买链接|g' README.md
```

## 第五步：测试购买流程

1. 用另一个账号或让朋友测试购买
2. 确认支付后能正常下载文件
3. 检查移动端体验
