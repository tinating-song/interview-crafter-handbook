# 知识星球配置指南

## 方案说明

由于面包多无法访问，我们使用**知识星球**作为销售平台。

## 快速方案（推荐）

### 使用微信赞赏码 + GitHub Issues

这是最快的启动方式：

#### 步骤1：创建收款码
1. 打开微信 → "我" → "服务" → "收付款"
2. 点击"赞赏码"
3. 保存赞赏码图片

#### 步骤2：创建购买指引页面

在GitHub仓库创建购买说明：

```markdown
# 购买完整版

## 价格：¥29

## 购买方式

1. 扫描下方微信赞赏码支付 ¥29
2. 支付后截图发邮件到：你的邮箱
3. 邮件标题注明：面试通关手册购买
4. 我们会在1小时内发送PDF到你的邮箱

## 微信赞赏码

[赞赏码图片]

## 包含内容

- 50道算法题解
- 10+系统设计案例
- 行为面试STAR模板
- 简历优化指南
- 100+面试官问题清单
```

#### 步骤3：更新README

将购买链接改为GitHub Issues链接

## 知识星球完整方案

### 优点
- 稳定可靠
- 支持持续内容更新
- 可以建立社群

### 缺点
- 需要微信公众号
- 提现需要一定手续费
- 需要审核时间

---

### 配置步骤

#### 步骤1：注册知识星球
1. 访问：https://wx.zsxq.com/
2. 下载知识星球App
3. 用微信注册/登录

#### 步骤2：创建星球
1. 点击"创建星球"
2. 填写信息：
   - 星球名称：面试通关手册
   - 星球分类：职场 / 技术交流
   - 付费模式：付费星球（¥29/人）
   - 星球简介：
   ```
   程序员技术面试速查手册

   包含：
   - 50道高频算法题解
   - 10+系统设计案例
   - 行为面试STAR模板
   - 简历优化指南
   ```
3. 提交审核（通常1-3个工作日）

#### 步骤3：审核通过后
1. 获取星球分享链接
2. 将PDF设置为置顶帖
3. 获取邀请链接
