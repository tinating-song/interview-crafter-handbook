## 🎯 STAR法则是什么？

**简单好记的回答框架**：

```
S - Situation 情境：我当时面临的背景

T - Task 任务：我需要完成的目标

A - Action 行动：我做了什么（重点！）

R - Result 结果：最终取得了什么成果
```

## 💬 50+高频PM面试问题

### 产品思维类（10+问）

**Q1: 如何判断需求优先级？**
A: 我用RICE模型...
- Reach（触达用户数）
- Impact（影响程度）
- Confidence（信心程度）
- Effort（开发成本）
加权打分，按分数排序

**Q2: 如何评估产品价值？**
A: 我从三个维度评估：
1. 用户价值：解决用户什么问题
2. 商业价值：带来多少收入/成本节省
3. 战略价值：是否符合公司方向

**Q3: 如何做trade-off决策？**
A: 我的决策框架：
1. 列出所有选项和约束
2. 分析每个选项的利弊
3. 咨询相关stakeholder
4. 做出决策并记录理由
5. 设定review时间点

### 项目管理类（10+问）

**Q4: 如何与开发协作？**
A: 三个关键实践：
1. 需求评审：PRD写清楚，开发前评审
2. 进度同步：每日站会同步风险和进度
3. 变更管理：变更走流程，不口头传达

**Q5: 如何与设计协作？**
A: 我的方法：
1. 设计前：参与用户调研，一起定义问题
2. 设计中：定期看进度，及时反馈
3. 设计后：参与用户测试，收集反馈

**Q6: 如何应对需求变更？**
A: 我的处理方式：
1. 理解变更原因：业务需求还是技术调整
2. 评估影响：涉及哪些模块、影响范围
3. 沟通协调：与开发/测试/设计评估
4. 优先级重排：是否影响当前sprint

### 数据分析类（10+问）

**Q7: 如何定义核心指标？**
A: 我用AARRR模型：
- Acquisition（获取）：用户来源和渠道
- Activation（活跃）：首次使用关键行为
- Retention（留存）：持续使用率
- Revenue（收入）：变现能力
每个阶段都有对应的北极星指标

**Q8: 如何做漏斗分析？**
A: 我的分析流程：
1. 定义漏斗各层级（曝光→点击→使用→付费）
2. 计算各层转化率
3. 找出流失最大的环节
4. 提出优化假设并A/B测试

**Q9: 如何通过数据驱动决策？**
A: 我的框架：
1. 提出假设："增加推荐入口会提升点击率"
2. 设计实验：对照组vs实验组
3. 收集数据：收集1周数据
4. 统计显著性：计算p-value < 0.05
5. 做出决策：显著则全量上线

### 行业理解类（10+问）

**Q10: 如何分析竞品？**
A: 我的分析框架：
1. 功能对比：列出核心功能差异
2. 用户体验：实际使用各产品，记录痛点
3. 商业模式：变现方式、定价策略
4. 技术实现：查看技术栈、评估壁垒

**Q11: 如何做市场调研？**
A: 我的方法：
1. 用户访谈：深度访谈10-15个目标用户
2. 问卷调查：更大样本量，量化验证
3. 数据分析：用行业报告验证趋势
4. 竞品分析：找到市场空白和机会

### 角色认知类（10+问）

**Q12: 产品经理 vs 项目经理？**
A: 我的理解：
- 产品经理：关注"做什么"和"为什么"
- 项目经理：关注"什么时候做完"和"谁来做"
- 两者互补，优秀PM需要兼备

**Q13: 数据产品 vs 策略产品？**
A: 差异点：
- 数据产品：依赖数据分析，指标驱动
- 策略产品：依赖行业洞察和战略方向
- 工作方式：数据产品更偏迭代，策略产品更偏规划

**Q14: AI产品 vs 传统产品？**
A: 核心差异：
- 不确定性：AI输出有随机性，需要质量评估
- 迭代速度：AI产品可以快速实验
- 用户期望：用户对AI的容错率更低
- PM能力：AI产品PM需要懂Prompt工程

## 📖 实战STAR模板示例

### 示例：介绍成功产品

```
【S】在上一家公司，我负责一个AI写作助手产品。
当时面临的问题是：用户反馈写作质量不稳定，满意度只有75%。

【T】我的目标是在2个月内提升用户满意度到85%，
同时降低内容生成的幻觉率。

【A】我采取了以下行动：
1. 数据分析：收集了1000+用户使用数据，
   发现问题主要集中在专业领域（法律/金融）
2. 竞品调研：对比了ChatGPT和Claude在专业领域的表现，
   Claude在专业知识上更有优势
3. 产品方案：切换到Claude API，并针对专业领域
   添加知识库（RAG方案）
4. A/B测试：先对10%用户灰度，
   监控满意度和幻觉率指标
5. 全量上线：2周内完成全量切换

【R】最终结果：
- 用户满意度从75%提升到88%（超出目标）✅
- 专业领域内容幻觉率从15%降低到3%✅
- 用户留存率提升12%
- 这个项目让我获得了季度最佳产品奖
```
